# D0 VM Cell-based v3 Experiment Results (Paper Edition)

## Architecture

- **Unified Cell system**: Code/Energy/Stomach cells with per-cell freshness
- **Two-step digestion**: EAT (food pool -> Stomach) + DIGEST (Stomach -> Energy)
- **Local REFRESH**: refreshes cells within radius R of current IP position
- **Gradual degradation**: individual cells die (freshness=0), not instant organism death
- **CEM=50** (cell energy max): required for DIVIDE to emerge
